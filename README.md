# Valley of Death

## Description

In this game you encanate Death himself in his daily job of recolecting the souls of the dead.

## Key Features

 - Colect Souls
 - Jump
 - Complete Levels
 
## Controls

 - A -> move left
 - D -> move right
 - SPACE -> jump
 - F1 -> reset character
 - F3 -> reset level
 - F9 -> Un/see colliders
 - F10 -> GOD mode
    - W -> Fly up
    - S -> Fly down
